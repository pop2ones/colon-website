# 特色功能

* 语音识别后的文本会自动进行分段，区分对白，添加标点
* 平铺的TXT显示方式更容易修改
* 可使用iCloud云盘保护数据安全
* 无需网络权限，开发者无法获取您的数据（iCloud云盘和语音识别会连接苹果公司网络）
